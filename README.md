Mini-Shop E-Commerce
A modern, responsive e-commerce frontend built with React and TypeScript. This project features a dynamic product catalog, advanced filtering, and a seamless shopping cart experience.
Key Features
Dynamic Catalog: Fetches real-time data from the DummyJSON API.

Quick View: Modal-based product previews for faster browsing.

Image Magnifier: High-quality zoom effect on product hover.

Cart Management: Full Redux-powered cart (Add, Remove, Update quantity).

Filtering: Filter products by category, price range, and ratings.

🛠️ Tech Stack
Frontend: React (TS), Tailwind CSS

State Management: Redux Toolkit

Routing: React Router Dom

API: DummyJSON
